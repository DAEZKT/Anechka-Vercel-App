# DAEZKT Branding Guidelines

## Identidad de Marca
- **Nombre de la Aplicación:** DAEZKT
- **Eslogan:** "We Simplify Your Tasks"
- **Tono de voz:** Profesional, Directo y Facilitador. La aplicación debe guiar al usuario de forma clara sin tecnicismos innecesarios, enfocándose en la eficiencia.

## Paleta de Colores
- **Color Primario:** `#8B5CF6` (Violeta Intenso - Acción principal, énfasis)
- **Color Secundario:** `#A78BFA` (Violeta Claro - Elementos secundarios, hovers)
- **Estilo Visual:** Glassmorphism / Light Gradient (Fondos con transparencias sutiles, desenfoques y gradientes suaves para una apariencia moderna y limpia).

## Reglas de Interfaz (UI/UX)
1. **Consistencia Visual:** Mantener el uso de componentes con estilo "Glass" (GlassCard) para contenedores principales.
2. **Feedback al Usuario:** Siempre mostrar indicadores de carga (spinners) durante operaciones asíncronas.
3. **Manejo de Errores:** Mensajes de error claros y accionables (ej: "No se encontró el producto" en lugar de "Error 404").

## Uso en Código
- Al referirse a la aplicación en títulos, metadatos o textos legales, usar siempre "DAEZKT".
- Los colores deben implementarse usando las variables de Tailwind o CSS definidas, preferiblemente extendiendo la configuración del tema si es necesario.
